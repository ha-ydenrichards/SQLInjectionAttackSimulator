# SQL Injection Attack Simulator
# This is an educational simulation that can be used to help students understand the basics of SQL Injection Attacks.
# For each level, the student is given a task to accomplish, as well as some necessary information needed to solve that problem.
# The task of the student is to log in to the database using the given criteria mentioned!
